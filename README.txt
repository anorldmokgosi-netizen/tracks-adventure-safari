Tracks Adventure Safaris & Lodges - Deploy Instructions

1. Create a GitHub repository and push these files to the `main` branch.
2. On Netlify, choose 'New site from Git' -> connect GitHub -> select the repo -> Deploy.
3. In Netlify Dashboard: Identity -> Enable Identity, then Services -> Git Gateway -> Enable Git Gateway.
4. In Netlify Dashboard: Forms -> you'll see form submissions from the booking form. Configure notifications to send to tracksadventuresafari2016@gmail.com.
5. Visit https://<your-site>.netlify.app/admin to access the CMS (create an Identity user or invite yourself).

Files included:
- index.html, safaris.html, about.html, terms.html
- styles/styles.css
- scripts/main.js
- admin/config.yml
- content/safaris/chobe-national-park.md
- netlify.toml, _redirects
- images/logo.png, images/hero.jpg
